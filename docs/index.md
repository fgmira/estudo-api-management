# Estudo API Management

Bem-vindo à documentação de estudos sobre **API Management**.

Este repositório reúne conceitos, padrões e boas práticas sobre gerenciamento de APIs.

## Como rodar localmente

```bash
poetry run mkdocs serve
```

Acesse em: [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Estrutura do projeto

```
mkdocs.yml        # Configuração do MkDocs
docs/
    index.md      # Página inicial
    ...           # Demais páginas
```
